# Arduino Sensor Backend

## About
- Receives sensor readings from an Arduino device with ESP32 Wifi module
- Exposes HTTP and Websocket endpoints to view the data

## Tech Stack
- FastAPI
- Websockets

## Deployment
Deployed on AWS Lambda and ECR
